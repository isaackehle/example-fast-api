# example-fast-api


## Purpose

This is a simple demo for a python FastAPI backend with NextJS frontend

RESTful endpoint for a new purchase order request

Fields:

POST /api/order
    - User
    - Item Name
    - Item Description
    - Item Cost
    - Reason For Purchase

nice to have:

GET /api/items
    - id
    - description
    - cost
    - name
